# 待办事项

## 待修复

### 1. 文件名 URL 编码处理
**问题：** 文件名存在空格或特殊字符时会导致请求报错

**影响范围：**
- 后端 API 路径参数处理
- 前端文件路径传递

**解决方案：**
- 后端：对文件路径进行 `decodeURIComponent()` 解码
- 前端：对文件路径进行 `encodeURIComponent()` 编码
- 或者统一使用请求体传递路径参数

**优先级：** 高

---

### 2. 权限错误处理
**问题：** 创建文件夹和压缩文件时出现权限不足错误

**影响范围：**
- `backend/src/routes/folders.js` - 创建文件夹
- `backend/src/routes/files.js` - 压缩文件

**解决方案：**
- 添加完善的 try-catch 错误处理
- 返回友好的错误提示信息
- 记录详细错误日志

**优先级：** 高
